# Flask Computer Equipment Management System

## Описание проекта
Система учета компьютерной техники на базе Flask. Приложение позволяет вести учет компьютерного оборудования, управлять пользователями с различными ролями доступа, отслеживать состояние техники и генерировать отчеты.

## Функциональные возможности
- Аутентификация и авторизация пользователей
- Разграничение прав доступа (администратор, менеджер, пользователь)
- Полный CRUD для оборудования и пользователей
- Фильтрация и сортировка данных
- Статистика по состоянию оборудования и пользователям
- Хранение подробной информации об оборудовании
- Защита паролей с помощью хэширования

## Технологии
- Flask (Python framework)
- SQLAlchemy ORM
- SQLite (база данных)
- Flask-Login для управления сессиями
- HTML/CSS/JavaScript для фронтенда
- pytest для тестирования
